# Calculator Web App

A simple and responsive calculator built using HTML, CSS, and JavaScript.

## Features

- Basic arithmetic operations (+, −, ×, ÷, %)
- Clear (AC) and Delete (DEL) functionality
- Input validation for safe calculations
- Responsive design

## Technologies Used

- HTML5
- CSS3
- JavaScript

##  Project Structure

calculator/
│── index.html
│── style.css
│── script.js

##  How to Use

1. Click number buttons to enter values.
2. Use operators for calculations.
3. Press "=" to get the result.
4. Use "AC" to clear everything.
5. Use "DEL" to remove the last digit.
